echo命令及转义字符（基于Bash）

基本用法

echo -n  DO NOT change line
     -e  parse escaping characters
         (escaping characters must be bordered by double quotation marks)

转义字符

转义字符 含义
\c 禁止换行
\t 水平制表符
\n 回车并换行
\\ 字面值\

实例

  • $ echo there is a table.
    there is a table.
  • $ echo -e Hello, "\n"cat!
    Hello,
    cat!
  • $ echo "there is a table."
    there is a table.

注意

如果使用echo --help只会得到：--help；这是因为echo会将其参数视为字符串。

如果想查看echo的帮助文档，可以使用以下命令：

  1. help echo
  2. info echo
  3. man echo
  4. env echo --help

通配符

通配符就是Shell中具有特殊用法的字符

通配符 含义（用法）
* 匹配任意数量的字符（包括零个）
？ 匹配任意一个字符
[...] 匹配[]中任意一个字符
[!...] 匹配不存在于[]中任意一个字符
{...} 匹配{}中任意一个字符串

实例

通配符在文件操作中十分有用。

元字符

元字符包含了转义字符和通配符以及其它特殊字符。

其他元字符：

元字符 用法
= 变量赋值
$ 变量或运算替换
< 输入重定向；将文件内容作为文件的输入
> 输出重定向；覆写输出内容到文件
>> 输出重定向；追加输出内容到文件
| 管道（pipeline）；将前一个命令的输出作为后一个命令的输入
; 命令分隔符；用来分隔不同命令使之按顺序执行
\ 转义下一个字符（如$, *, <, >等），使之解释为字面值
`` 重音符；首先执行其中的命令，将输出作为字符串插入
& 将程序放到后台执行，不阻塞终端
() 在子Shell中执行命令组、命令替换、数组初始化
&& 继续执行下一条指令，当且仅当上一条命令返回true
|| 继续执行下一条指令，当且仅当上一条命令返回false

消除元字符的特殊含义

Shell中提供了消除元字符、通配符含义的字符

  • \ ：其后的元字符使用字面值
  • ""（双引号） ：其中的元字符（ $ , ``（重音符）和 \ 除外 ）使用字面值
  • ''（单引号） ：其中的元字符全部使用字面值

实例

  • echo temp? 将文件夹中以temp为开头，且其后跟着单个字符的文件名打印出来
  • echo temp\? 打印temp?
  • echo \< \> \? \$ \' \" \| \\ 打印< > ? $ ' " | \
