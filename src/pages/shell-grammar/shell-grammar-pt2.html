本次更加偏重流程控制：

条件控制

基本语法

[ condition ]

注意

  1. condition与[]不能相连，其中有空格。
  2. 条件非空为true,[]返回false。

常用判断条件

整数之间比较：

    -lt less than
    -le less or equal
    -eq equal
    -gt greater than
    -ge greater or equal
    -ne not equal

字符串之间比较：

    = equal
    != not equal

文件权限的判断：

    -r read
    -w write
    -x execute

文件类型的判断：

    -f exist a normal file
    -e existence
    -d a directory

实例

  1. 23是否大于等于22？
    $ [ 23 -ge 22 ]
    $ echo $?
    0

  2. 判断hello_shell.sh是否具有写权限？
    $ [ -w hello_shell.sh ]
    $ echo $?
    0

多条件判断

[ condition1 ] && [ condition2 ] || [ condition3 ]

if判断

基本语法

if [ condition ];then
  # coding here
fi

也可以这样写： 
if [ condition ]
  then
    # coding hear
fi

for循环

基本语法1

for ((初始值;循环控制条件;变量变化))
do
  # coding here
done

注意

for与(有空格！！！
(与条件之间没有任何空格字符！！！

基本语法2

for var in val1 val2 val3...
do
  # coding here
done

实例

  1. 计算从1加到100的值

    #!/bin/bash

  s=0
  for ((i=1;i<=100;i++))
  do
    s=$[$s+$i]
  done

  echo $s

  2. 打印所有参数

    #!/bin/bash

for i in $@
do
  echo $i
done

注意

在2中，$* = $@ != "$*"(as a whole)

while循环

基本语法

while [ condition ]
do
  # coding here
done

注意

while与[之间有空格；并且[与条件之间同样有空格！！！

实例

计算从1加到100的值

#!/bin/bash

s=0
i=1
while [ i -le 100 ]
do
  s=$(($s+$i))
  i=$(($i+1))
done

echo $s
